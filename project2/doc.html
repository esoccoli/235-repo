API Info

For this project, I used the PokeAPI to display data about various Pokemon. The specific endpoint I used for this data was Pokemon. This endpoint returns a wide variety of data, and I took that, parsed it into a JSON, and then displayed the following information on the page:

  • Name
  • Pokedex Number
  • Type(s)
  • Height
  • Weight
  • Potential Abilities
  • Base Stats
  • Possible Moves
