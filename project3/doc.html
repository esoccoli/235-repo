Process

My process for working on this project was breaking the project into many smaller parts, and working on one part at a time, and then moving to the next part once the current one is complete. For example, I started by doing the scene setup, which included creating all the scenes, and adding all necessary text and buttons to the scenes. After that, I started working on the game itself. For that, I started by making the paddle and getting it to move left and right smoothly with user input. Then, I started working on the ball, making it move around the screen and bounce off walls and the paddle. Once that was working sufficiently, I made the bricks, and added collisions between the ball and the bricks. After that, I implemented power ups. Once all of that was implemented, I went back and started fixing bugs that had came up while developing.

Sound Effects

My game has sound effects that play when the following events occur:

  • Brick gets broken
  • Power up collected
  • Life lost
  • Level cleared
  • Game over

Other Requirements

Listed below are many of the other requirements that I met

  • All CSS and JS is in separate files
  • Multiple JS files are used to separate concerns
  • All variables are declared using let or const
  • Repeated code is separated out into functions
  • All HTML and CSS passed validation
  • Code is well commented
